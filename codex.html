01 / INSTALACIÓN

Instalá la CLI

Necesitás Node.js. Instalá Codex globalmente y comprobá que el binario quedó disponible.

npm install --global @openai/codex
codex --version

También existe un instalador independiente para macOS y Linux. Elegí un método y mantenelo actualizado desde el propio CLI.

02 / PRIMER INICIO

Autenticate y trabajá en un proyecto

1

Entrá a la carpeta de tu proyecto y ejecutá codex.

2

Elegí iniciar sesión con tu cuenta de ChatGPT o con una API key de OpenAI. La CLI conserva tus credenciales de forma local.

3

Describí una tarea concreta, por ejemplo: revisá la arquitectura de este repositorio.

cd mi-proyecto
codex
Configuración personal y por proyecto

Tus preferencias viven en ~/.codex/config.toml. Para compartir ajustes no sensibles con un repositorio, usá .codex/config.toml dentro del proyecto confiable.

03 / PROVEEDOR REMOTO

Usar OpenRouter

OpenRouter expone una API compatible con Chat Completions. Creá una API key en su panel, exportala sólo en tu sesión y declaralo como un proveedor personalizado.

export OPENROUTER_API_KEY="tu_clave_aquí"

Agregá este bloque a ~/.codex/config.toml y reemplazá el modelo por un slug vigente del catálogo de OpenRouter.

model = "anthropic/claude-sonnet-4"
model_provider = "openrouter"

[model_providers.openrouter]
name = "OpenRouter"
base_url = "https://openrouter.ai/api/v1"
env_key = "OPENROUTER_API_KEY"
wire_api = "chat"
Antes de usarlo en trabajo crítico

Las capacidades de herramientas, razonamiento y streaming dependen del modelo y de la compatibilidad del proveedor. Probá una tarea pequeña primero y mantené la clave fuera del repositorio.

04 / MODELOS EN TU EQUIPO

Ejecutar con Ollama

Codex ofrece modo OSS para proveedores locales. Con Ollama instalado y un modelo de código descargado, iniciá una sesión con --oss.

ollama pull qwen3-coder
codex --oss --local-provider ollama

Para dejar Ollama como opción local predeterminada, añadí esta línea a tu config.toml:

oss_provider = "ollama"
Consejo de rendimiento

Los agentes usan bastante contexto y herramientas. Elegí un modelo con capacidad suficiente para tu memoria/RAM y verificá que soporte tool calling antes de depender de él.
