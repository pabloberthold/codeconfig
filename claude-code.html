01 / INSTALACIÓN

Instalá Claude Code

Con Node.js 18 o superior podés instalar la CLI globalmente. Evitá usar sudo con npm global: puede generar problemas de permisos.

npm install -g @anthropic-ai/claude-code
claude doctor

En macOS, Linux y WSL también está disponible el instalador nativo en evolución. Consultá la página oficial para elegir el método recomendado para tu plataforma.

02 / PRIMER INICIO

Iniciá sesión y empezá a trabajar

1

Abrí tu proyecto desde una terminal y ejecutá claude.

2

Elegí autenticarte con Anthropic Console, o con tu plan Claude Pro / Max cuando esté disponible para tu cuenta.

3

Pedile una primera tarea enfocada: explicá la estructura de este proyecto y sugerí por dónde empezar.

cd mi-proyecto
claude
Para automatizar

El modo no interactivo usa claude -p "tu instrucción". Agregá límites de turnos y permisos explícitos antes de incorporarlo a CI.

03 / RUTAS SOPORTADAS

Elegí el proveedor adecuado

Claude Code se conecta de forma nativa a la API de Anthropic. Para organizaciones, la documentación oficial contempla Amazon Bedrock y Google Vertex AI con sus credenciales y variables de entorno correspondientes.

# Ejemplo de Amazon Bedrock
export CLAUDE_CODE_USE_BEDROCK=1
export AWS_REGION=us-east-1
claude

Usá esta ruta si tu organización ya centraliza modelos Claude en AWS o Google Cloud. No requiere que expongas claves de Anthropic a cada estación de trabajo.

Sobre OpenRouter y Ollama

Claude Code no documenta una conexión directa y nativa a OpenRouter u Ollama. Para esos proveedores, la alternativa es un gateway que exponga una API compatible con Anthropic. Tratala como una integración avanzada y validala en un entorno aislado.

04 / INTEGRACIÓN AVANZADA

Usar un gateway compatible

Un gateway centraliza autenticación, routing y observabilidad entre Claude Code y otros proveedores. Anthropic documenta el uso de gateways compatibles con el formato de su API; LiteLLM es un ejemplo de tercero.

1

Desplegá y asegurá el gateway según la documentación de ese producto. Configurá allí el acceso al proveedor que necesites.

2

Apuntá Claude Code al endpoint compatible y entregale el token del gateway mediante variables de entorno.

export ANTHROPIC_BASE_URL="https://tu-gateway.example.com"
export ANTHROPIC_AUTH_TOKEN="token-del-gateway"
claude
Importante

El gateway debe implementar el formato de Anthropic y ofrecer los modelos/capacidades que Claude Code necesita. OpenRouter y Ollama no son sustitutos directos del endpoint de Anthropic; no copies este bloque sin haber configurado antes esa capa intermedia.
