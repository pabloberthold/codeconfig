OpenCode

Un agente de terminal abierto y proveedor-agnóstico. Conectalo en pocos pasos a OpenRouter, Ollama local o cualquiera de los proveedores compatibles.

ENTORNOmacOS · Linux · Windows
CONFIGURACIÓNopencode.json

01 / INSTALACIÓN

Instalá OpenCode

El instalador oficial es la forma más directa. También podés usar npm si ya administrás las herramientas globales con Node.js.

curl -fsSL https://opencode.ai/install | bash
# alternativa: npm install -g opencode-ai

Verificá la instalación abriendo una carpeta de trabajo y ejecutando opencode.

02 / PRIMER INICIO

Conectá un proveedor desde la interfaz

1

Iniciá opencode en el directorio de tu proyecto.

2

Escribí /connect, elegí el proveedor y pegá la clave cuando la pida.

3

Usá /models para seleccionar el modelo que vas a utilizar.

cd mi-proyecto
opencode
# dentro de OpenCode: /connect
03 / PROVEEDOR REMOTO

Conectar OpenRouter

OpenCode incluye OpenRouter entre sus proveedores. Creá la key en OpenRouter y seguí el flujo interactivo: no hace falta escribir la clave en un archivo de configuración.

opencode
# dentro de OpenCode:
/connect
# Elegí “OpenRouter” y pegá tu API key
/models

Los modelos habituales ya aparecen en el selector. Si necesitás registrar uno extra o ajustar opciones de routing, creá o editá opencode.json:

{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "openrouter": {
      "models": { "moonshotai/kimi-k2": {} }
    }
  }
}
04 / MODELOS EN TU EQUIPO

Usar Ollama local

La integración oficial de Ollama puede preparar OpenCode y pasar la configuración automáticamente. Requiere Ollama instalado y un modelo apto para agentes.

ollama launch opencode

Para generar la configuración sin iniciar OpenCode, agregá --config. Si preferís declararlo vos mismo, OpenCode acepta un endpoint compatible con OpenAI:

{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "ollama": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "Ollama (local)",
      "options": { "baseURL": "http://localhost:11434/v1" }
    }
  }
}
Ventana de contexto

Para tareas de código largas, la integración de Ollama recomienda una ventana de contexto amplia. Ajustala según el modelo y la memoria disponible.
