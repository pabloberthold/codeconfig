# Agent Atlas

Sitio estático en español para centralizar la instalación y configuración de Codex, OpenCode y Claude Code.

## Publicar con GitHub Pages

1. Subí estos archivos a la rama que uses para publicar (por ejemplo, `main`).
2. En GitHub, abrí **Settings → Pages**.
3. En **Build and deployment**, seleccioná **Deploy from a branch**, la rama y la carpeta raíz (`/`).
4. Guardá los cambios: GitHub mostrará la URL pública cuando termine el despliegue.

No requiere compilación ni dependencias: `index.html` es la portada.
